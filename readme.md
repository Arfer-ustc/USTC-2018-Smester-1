##	Tips

*	教辅系统无法登陆的问题

	mis给出的解决方案是，每次都把 `url` 前面的 `ssea.ustcsz.edu.cn` 手动换成 `219.219.220.50`

	更加便捷的方法是修改 `host` 文件, 这样就能做到自动跳转了

	*	Windows

		打开 C:\Windows\System32\drivers\etc\hosts 文件，在文件最后添加如下内容后保存

		```
		219.219.220.50	ssea.ustcsz.edu.cn
		```

	*	Linux / macOS

		首先打开 hosts 文件

		```
		$ vi /etc/hosts
		```

		然后在文件末尾添加如下内容

		```
		219.219.220.50	ssea.ustcsz.edu.cn
		```
