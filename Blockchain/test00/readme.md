##	区块链第一次实验

通过区块链课程的实验，我们将实现一个简化版的区块链，并基于它来构建一个简化版的加密货币。

第一次实验有以下几点内容，每一小节分别对应一个文件夹，其中包含了 `Go` 和 `C++` 的实现

*	Prototype, 基本原型

*	PoW, 工作量证明

*	Persistence, 持久化(内存->磁盘)

*	CLI, 命令行接口
